Python re模块简明参考表

常用函数

函数 描述 返回值
re.compile(pattern) 编译正则表达式 Pattern对象
re.search(pattern, string) 搜索第一个匹配 Match对象或None
re.match(pattern, string) 从开头匹配 Match对象或None
re.findall(pattern, string) 查找所有匹配 字符串列表
re.finditer(pattern, string) 查找所有匹配 Match对象迭代器
re.sub(pattern, repl, string) 替换匹配文本 新字符串
re.split(pattern, string) 按模式分割 字符串列表

常用特殊字符

字符 描述 示例
. 匹配任意字符(除换行符) a.c → “abc”
^ 匹配字符串开始 ^abc
$ 匹配字符串结束 abc$
* 0次或多次重复 a*b → “b”, “ab”, “aab”
+ 1次或多次重复 a+b → “ab”, “aab”
? 0次或1次重复 a?b → “b”, “ab”
{m,n} m到n次重复 a{2,4} → “aa”, “aaa”, “aaaa”
[] 字符集合 [abc] → “a”或”b”或”c”
| 或操作 a|b → “a”或”b”
() 分组 (abc)+

常用特殊序列

序列 描述 等价于
\d 数字 [0-9]
\D 非数字 [^0-9]
\w 单词字符 [a-zA-Z0-9_]
\W 非单词字符 [^a-zA-Z0-9_]
\s 空白字符 [ \t\n\r\f\v]
\S 非空白字符 [^ \t\n\r\f\v]
\b 单词边界 -
\B 非单词边界 -

常用标志(flags)

标志 描述 内联形式
re.I / re.IGNORECASE 忽略大小写 (?i)
re.M / re.MULTILINE 多行模式 (?m)
re.S / re.DOTALL 点匹配所有字符 (?s)
re.X / re.VERBOSE 详细模式(可加注释) (?x)
re.A / re.ASCII 仅ASCII匹配 (?a)

Match对象常用方法

方法 描述 示例
.group() 返回匹配的字符串 m.group()
.groups() 返回所有分组的元组 m.groups()
.start() 匹配开始位置 m.start()
.end() 匹配结束位置 m.end()
.span() (开始,结束)元组 m.span()

Pattern对象常用方法

方法 描述
.search(string) 搜索匹配
.match(string) 从开始匹配
.findall(string) 查找所有匹配
.finditer(string) 查找所有匹配(迭代器)
.sub(repl, string) 替换
.split(string) 分割

常用示例

重要提示

  1. 使用原始字符串: r"pattern" 避免转义问题

  2. 编译复用: 多次使用同一模式时先编译

  3. 匹配 vs 搜索: match()从开头，search()任意位置

  4. 贪婪 vs 非贪婪: *?、+?、?? 为非贪婪匹配
